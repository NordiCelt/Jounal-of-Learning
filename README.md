| --------- | ------------------ | ------------ | ---------------- | ----------------- | ----------------- |
|     7     |     Application    |  http, smtp  | User Application |    Application    |    Application    |
|     6     |     Presentation   |  jpg, ascii  |                  |                   |                   |

|     4     |     Transport      |  TCP or UDP  |     Firewalls    |     Transport     |     Transport     |
|     3     |     Network        |   IP, ICMP   |      Routers     |      Internet     |      Internet     |
